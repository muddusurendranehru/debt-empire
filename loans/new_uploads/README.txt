DRAG FILES HERE
===============
Drop PDF, DOCX, or Excel files here for loan verification.

• Bajaj PDF (e.g. bjmagnarepay1.pdf)  → py loan_verifier.py → Option 1 → 1 → type filename
• L&T / HDFC / other docs            → same: verify new loan, then enter filename

The verifier looks in: this folder, project root, and Documents.
