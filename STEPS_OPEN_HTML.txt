  OPTION A ‚Äî WITHOUT SERVER (simplest, no PowerShell):
    1. Open your PC and open Chrome or Edge (same browser you used before).
    2. Open File Explorer.
    3. In the address bar paste: c:\Users\MYPC\Desktop\debt-arbitrage\debt-empire
    4. Double-click: verifier_with_upload_v3.1.html  (v1.5, preferred) or verifier_with_upload_v3.2.html or senior_arbitrage_manager.html
    5. The app opens. Your loans are still there (same browser = same data).
    ‚Üí To open the app next time: repeat steps 2‚Äì4. No server needed.

  OPTION B ‚Äî WITH SERVER (if you use http://localhost:8080/...):
    1. Open your PC and open Chrome or Edge (same browser you used before).
    2. Open PowerShell.
    3. Run: cd c:\Users\MYPC\Desktop\debt-arbitrage\debt-empire
    4. Run: python -m http.server 8080
    5. In the browser ADDRESS BAR (not Google search) type: http://localhost:8080/verifier_with_upload_v3.1.html  (v1.5 / preferred)
       Or v3.2: http://localhost:8080/verifier_with_upload_v3.2.html
       Or Senior Manager: http://localhost:8080/senior_arbitrage_manager.html
       Or Editable Master Dashboard: http://localhost:8080/editable_master_dashboard.html
    6. Press Enter. The app opens. Your loans are still there.
    ‚Üí To open the app next time: repeat steps 2‚Äì6. Start the server first (steps 2‚Äì4).

  IMPORTANT:
  ‚Ä¢ Use the SAME way every time: either always ‚Äúdouble-click file‚Äù OR always ‚Äúlocalhost:8080‚Äù. Data is stored per ‚Äúorigin‚Äù (file vs localhost), so if you switch, you may see 0 loans until you use the same way again.

DEMO FLOW ‚Äî ADD A DEMO LOAN AND TRY ALL 3 STEPS
------------------------------------------------

STEP 1 ‚Äî DOCUMENT EXTRACTOR (Tab 1)
  1. Stay on Tab 1: DOCUMENT EXTRACTOR.
  2. Drag a PDF/DOCX onto the drop zone (or click and pick a file).
  3. Wait for "Done. Click ‚ûï ADD TO DASHBOARD or EXTRACT GOOD ‚Üí NEXT."
  4. Check/correct the extracted fields (loan name, account, OS, EMI, etc.).
  5. Either:
     ‚Ä¢ Click ‚ûï ADD TO DASHBOARD ‚Üí loan is added; go to Tab 3 to see it.
     ‚Ä¢ Or click ‚úÖ EXTRACT GOOD ‚Üí NEXT: VERIFIER ‚Üí goes to Step 2.

STEP 2 ‚Äî LOAN VERIFIER (Tab 2)
  1. Tab 2 shows existing loans in the dropdown and "+ NEW LOAN."
  2. Select "+ NEW LOAN" (or an existing loan to edit).
  3. Fill/verify: Loan Name, Account Number (required), Borrower, OS, EMI, Disbursal, Loan Type.
  4. Click ‚úÖ SAVE LOAN.
  5. Account is required; if empty you get: "‚ö†Ô∏è Account number required. Check document header."
  6. After save: "‚úÖ Loan xxx saved! Run 'py empire.py' to refresh dashboard."
  7. Tab 3 list and main dashboard refresh automatically.

STEP 3 ‚Äî DEBT MANAGER (Tab 3)
  1. Tab 3 shows active loans list (name, account, OS, EMI, docs count).
  2. Select a loan from the dropdown to attach new docs (drag zone + doc type buttons).
  3. Live summary shows: "‚úÖ X loans active | Total Exposure: ‚ÇπY."
  4. Click OPEN FULL DASHBOARD to open verifier.html in a new tab (optional).
  5. Run py empire.py in PowerShell only if you want to update an external dashboard file.
